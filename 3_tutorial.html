Intro to Heads-up Display


The heads up display is a transparent image that features important information on the windshield such as the speed limit and the driver’s current speed. When automation is on, the display expands to show a meter which chronicles the duration of autodrive and interactive applications below allowing the user to utilize the windshield while having view of the road.
