Mobile Office Study

Task Overview


In this study, we're investigating how drivers could engage with non-driving tasks while the car’s autopilot is on.

We will present you with two scenarios: a non-driving work-related activity and a non-driving leisure-related one.

For each scenario, we will show you a series of images of an in-vehicle user interface. We will then ask you to show us which user command would result in the user interface looking as it does in the image.

For each image, we will ask you to show us (using video recording) which voice or gesture command could result in the user interface depicted in the image.
