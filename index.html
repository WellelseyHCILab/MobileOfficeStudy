Mobile Office Study

Introduction


Welcome!

Thank you for participating in this research study.

The purpose of this study is to investigate user preferences when interacting with an environment for future, highly-automated cars.

We will show you in-vehicle user interface designs, which support work-related and leisure tasks, as well as safe driving.

This study is part of a larger National Science Foundation (NSF) funded project about The Mobile Office: Future of Work in Highly-Automated Cars.

More information about the project and the investigators can be found on our website: Mobile Office.
